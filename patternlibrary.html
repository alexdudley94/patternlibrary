Pattern Library

Navigation

  • Navigation Bar

    • The header-nav must be present on every page.

    The header-nav must also always be inside the header element. The header-login must always be present whenever header-nav is used.


  • Header

    • The header class creates the white bar and must be present on every page.

    Logo class assigns the logo with automatic positioning. This must not be altered, the logo must remain unedited and cannot be covered by any other element under no circumstances. Header-nav and header-login must always be present within the header.

  • Jumbotron

    • The jumbotron class. Can be used on descriptive pages only, e.g about.html

  • Footer

    • The footer class must always include the social media links.












  • Buttons

  • Submit Buttons

    • Only for submission elements.



  • Interactive Buttons

    • For interactive elements e.g open/close menu.


  • Forms



    • Forms must be uniformed and have rounded corners. Labels must always be present, placeholder cannot be used.
